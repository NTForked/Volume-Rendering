Volume Rendering using Raycasting

Made with **C++**, **OpenGL**, **Cinder**, **GLM** and **dear imgui**.

*Requirements:*

**OpenGL 4.3+**

Utilises compute shaders for fast gradient, gradient smoothing and histogram computation.
